# 🚀 C Programming Lab & Practice Programs

Welcome to the **C-lang Repository**!
This repository contains a collection of **C programming programs** created for **learning, practice, and academic lab work**.

It covers important C concepts such as **recursion, file handling, memory management, and command line arguments**.

---

## 📚 Topics Covered

✔ Basic C Programs
✔ Fibonacci Series using Recursion
✔ String Operations
✔ File Handling (`fopen`, `fprintf`, `fscanf`)
✔ File Pointer Functions (`ftell`, `fseek`, `rewind`)
✔ Command Line Arguments
✔ Dynamic Memory Allocation (`malloc`, `calloc`, `free`)

---

## 🧑‍💻 Technologies Used

* **C Programming Language**
* **GCC Compiler**
* **Standard C Libraries**

## ⚙️ How to Run the Programs

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/mrbhavikpandit/C-lang.git
```

### 2️⃣ Open the Folder

```bash
cd C-lang
```

### 3️⃣ Compile the Program

```bash
gcc program_name.c -o program
```

### 4️⃣ Run the Program

```bash
./program
```
